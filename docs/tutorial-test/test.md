## Примеры кода

Ниже приведены примеры кода в различных языках программирования :

```javascript
// Пример JavaScript кода

	return true;

```

```python
# Пример Python кода
def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n-1)
```

```rust
// Пример Rust кода
fn main() {
    println!("Привет, мир!");
    let x = calculate_something();
    println!("Результат: {}", x);
}

fn calculate_something() -> i32 {
    let mut result = 0;
    for i in 1..10 {
        result += i;
    }
    result
}
```
